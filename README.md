# My Stocks

## Overview

### Key features
- Login / Sign up / Logout
- Show stock news / daily stock information
- Users can choose stock and add to watchlist
- Closer look at single stock

### Technologies used
#### Frontend
- HTML
- CSS
- React
- Redux 
- Redux-thunk
- React-vis
- React Router
- Hooks

#### Backend
- Node.js
- Express.js
- MongoDB
- Bcrypt.js
- JSON Web Tokens
- Multer
- Mongoose Schema
